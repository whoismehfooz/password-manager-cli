# 🔐 Password Manager CLI

A simple yet powerful **Command Line Password Manager** built with Python.
Store, manage, and retrieve your credentials directly from the terminal — fast, clean, and efficient ⚡

---

## 🚀 Features

* ➕ Add new passwords
* 📖 View all saved credentials
* 💾 Persistent storage using JSON
* 🧠 Clean CLI-based interaction
* 🗂 Organized project structure

---

## 🛠 Tech Stack

* Python 🐍
* JSON (for storage)
* OS module

---

## 📂 Project Structure

```
password-manager-cli/
│
├── main.py          # Entry point (CLI menu)
├── utils.py         # Core logic (add/view)
├── storage.py       # File handling (load/save)
├── requirements.txt
│
└── data/
    └── passwords.json
```

---

## ▶️ How to Run

```bash
python main.py
```
